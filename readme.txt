Задача:
	"Морской бой";
Автор:
	Левагин Данил, КН-204(МЕН-290201);
Описание:
	Программа представляет из себя реализацию игры "Морской бой".
	Информация по окнам:
	• В окне "Конфигурация" выбирается настройка игры(размер поля,
	  вкл./откл. подуровеня подводных лодок, сложность AI).

	• В окне "Расставление кораблей" вы можете разместить свои корабли
	  на поле, данные о количестве оставшихся для расстановки кораблей
	  находятся в левом верхнем углу; нажав на кнопку "DEL", можно перейти
	  в режим удаления поставленных кораблей, нажмите на неё второй раз,
	  чтобы выйти; нажав на кнопку "перейти на ... уровень"(доступна
	  лишь при включенной функции 3D-поля), вы перейдёте на другой уровень
	  (верхний/нижний) расстановки кораблей, нажмите на неё дважды, чтобы
	  вернуться; "джойстик" снизу позволяет перемещаться по полю(при размерах > 16x16);
	  когда вы расставите все возможные корабли, станется доступна
	  кнопка "начать игру"

	• В окне "Игра" происходит основное действие. Информация о количестве
	  оставшихся суден противника находится в левом верхнем углу; небольшая сводка
	  по игре отображается в центре экрана; для стрельбы нажимайте на ячейки поля
	  противника(справа); кнопка "перейти на ... уровень" по своему функционалу
	  аналогична кнопке из окна расстановки кораблей; "джойстик" снизу позволяет
	  перемещаться по полю(при размерах > 16x16).

	Игра поддерживает:
	1) Поле больших размеров(до 512x512- точно);
	2) 3D-поле(подуровень подводных лодок на игровом поле);
	3) Уровни сложности AI:
		1. Very easy- стрельба по рандомным точкам;
		2. Easy- противник научился не стрелять в точки, окружающие уничтоженный корабль;
		3. Normal- уровень Easy+ "добивание" большого корабля противника,
		   если ранее AI уже попал в какую-то его часть.
